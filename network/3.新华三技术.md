



# 操作系统

Comware是H3C自主研发的一款应用于网络设备的网络操作系统，它包含丰富的网络功能，是实现网络设备高性能、高可靠性的基础。

Comware支持从SOHO到数据中心、从企业级到运营商级的全系列网络设备。

最早期使用的是V5系统，现阶段主流用到的V7，以及最新的V9系统。现阶段能够用的系统架构一般的V7较多，在一些新设备上会使用V9系统。

# 堆叠

新华三中的堆叠技术，也叫IRF，IRF（Intelligent Resilient Framework，智能弹性架构）是H3C自主研发的软件虚拟化技术。

它的核心思想是将多台设备通过IRF物理端口连接在一起，进行必要的配置后，虚拟化成一台“分布式设备”。

使用这种虚拟化技术可以集合多台设备的硬件资源和软件处理能力，实现多台设备的协同工作、统一管理和不间断维护。
